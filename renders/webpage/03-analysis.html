Main effect, mediation, and moderation

Published

May 30, 2026

Three questions, three models — all defined once in R/functions/analysis.R and reused by the manuscript.

Main effect: does the frame move purchase intention?

Code 
fit_main <- fit_main_model(clean)
knitr::kable(broom::tidy(fit_main, conf.int = TRUE), digits = 3)

The combined_frame coefficient is the one to watch — a clear bump over the control reference.

Mediation: does perceived value carry the effect?

Three regressions tell the conceptual mediation story (Baron and Kenny 1986; Hayes 2017):

Code 
med <- fit_mediation_models(clean)
dplyr::bind_rows(
  total       = broom::tidy(med$total, conf.int = TRUE),
  mediator    = broom::tidy(med$a,     conf.int = TRUE),
  outcome_w_m = broom::tidy(med$b,     conf.int = TRUE),
  .id = "model"
) |>
  dplyr::filter(grepl("condition|perceived_value", term)) |>
  knitr::kable(digits = 3)

Read it like this: the condition coefficients shrink from total to outcome_w_m, and perceived_value is a strong predictor — the mediation signature.
